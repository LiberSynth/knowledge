Компиляция

1. Установить Java Development Kit (JDK)
2. Переменные среды - Системные переменные - Создать:
Имя: JAVA_HOME
Значение: <путь к установленной JDK>\bin
3. Переменная Path - Изменить - Создать:
Значение: %JAVA_HOME%\bin
4. Перезагрузка.
5. Проверка - в КС javac или javac -version

6. Переходим в каталог с программой
7. javac <main class>.java
8. Создается файл <main class>.class

Запуск скомпилированной программы

1. Перемещаемся на уровень выше.
2. java <каталог с классом = имя пакета>.<main class>

Конечно же, этот модуль будет пакетом (package в начале), потому что в нем понадобится import. Ну и, имя пакета должно быть такое же, как имя каталога, в котором лежит модуль, иначе ничего не заработает.
